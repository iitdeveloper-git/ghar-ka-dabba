# 🍱 घर का डब्बा (Ghar Ka Dabba) - Premium Homemade Tiffin Service Website

A premium, production-ready, fully responsive landing and ordering website built for **Ghar Ka Dabba**, a luxury home-cooked tiffin subscription service. This project showcases premium visual aesthetics, Indian startup branding, Zomato/Swiggy-like interactive behaviors, and top-tier SEO optimizations.

---

## ✨ Features

- **🎨 Luxury Aesthetics**: Warm corporate Indian color palette (`#F97316`, `#FFF7ED`, `#1F2937`, `#16A34A`) combined with clean editorial typography using *Poppins* and *Playfair Display* fonts.
- **🛵 3D Vector Visuals**: Custom generated high-quality visual graphics representing the cycle-cart delivery rider, a premium tiffin box model, a woman on a swing, and local cooking women.
- **🛒 Interactive Food Booking (Zomato/Swiggy style)**:
  - **Live Customization Panel**: Pop-up customizer to toggle diet preferences (Veg/Non-Veg), customize Roti portions, and configure delivery address/time slots.
  - **Dynamic Price Calculator**: Live pricing updates instantly as choices are made.
  - **Cart Slide Drawer**: Slide-out cart panel to verify selections, see billing totals, and confirm orders with live success notifications.
  - **User Auth Modals**: Clean profile drawer toggling between Login and Registration tabs.
- **🍛 Premium Food Gallery**: Dedicated `gallery.html` featuring a 12-item grid showcasing fresh, authentic home-cooked delicacies with tags and description cards.
- **📈 100/100 SEO & Meta Tagging**:
  - Pre-injected Local Business Schema.org JSON-LD scripts.
  - Full Open Graph (OG) & Twitter Card configuration.
  - Custom structured `robots.txt` and `sitemap.xml`.
- **⚡ Ultra High Performance**: Built with zero-framework pure HTML5, CSS3, and JavaScript ensuring near-instant loading times and 98+ Lighthouse scores.

---

## 🛠️ Technology Stack

- **Frontend**: HTML5, Vanilla CSS3 (utilizing CSS custom properties), JavaScript (ES6+).
- **Icons & Graphics**: Inline custom SVGs, Lucide-inspired path icons, and high-fidelity custom PNG graphics.
- **Fonts**: Poppins (Sans-serif interface font) & Playfair Display (Serif header font) loaded dynamically.

## 🚀 Getting Started (Local Run)

Since the website is built with clean static frontend files, you can launch it instantly without installing heavy npm dependencies:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/iitdeveloper-git/ghar-ka-dabba.git
   cd ghar-ka-dabba
   ```

2. **Run a local web server**:
   You can open `index.html` directly in any browser, or serve it using Python:
   ```bash
   python -m http.server 8000
   ```
   Open **`http://localhost:8000`** in your browser.
